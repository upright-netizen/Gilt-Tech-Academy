Let's start a project

This week we're going to start a project that you will continue to build on for the next 5 weeks. I suggested a portfolio for the work you've done a Gilt. But if that's not your jam, you can start a web site about anything! It does need to be something that we can incorporate all the concepts of this class into, which are document structure (paragraphs, headings, sections, lists, and maybe tables), images, links (so, it's going to need multiple pages), and lots of pretty CSS interactions.

For this week, I'd like to get started with one simple page that will act as the landing page for your project. I'd like you to use the following tags in some form or fashion before next week.

    Tags to use:
  • html, head, body
  • header, footer, section, article(optional), div
  • At least one list
    • ul, li
    • ol, li
    • dl, dt, dd
  • At least three of these
    • strong
    • em
    • blockquote
    • q
    • abbr
    • cite
    • dfn
    • address
    • ins
    • del
    • s, strike
