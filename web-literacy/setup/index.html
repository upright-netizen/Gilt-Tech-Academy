The Setup

Browser

  • Do you have Chrome?
  • If not, Download It
  • … and install it

Text Editor

  • Download Sublime Text 2
  • You know, install
  • symlink for command line support sudo ln -s /Applications/Sublime\ Text\ 2.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

Github Account

  • go to Github and sign up for a FREE account (if you don’t have one already)
  • Tell Nathan your username

Github app

  • Download the App
  • Install
  • Login
  • Choose ‘upright-netizen’
  • Clone Sesame Street
  • Create a branch for yourself
  • Sync your branch

Node

  • Download NodeJS
  • Install
  • Open Terminal
  • type node
  • At prompt, type 2 + 2
  • If you get 4, move on. If not raise your hand.

All together now

  • open Terminal
  • cd [path to sesame street]
  • node app/app.js
  • press Ctrl+C
  • create an alias 
     # This is your terminal
    $> cd ~;
    $> touch .profile;
    $> subl .profile;
    #paste this in
    alias sesame=’node ~/Code/Sesame-Street/app/app.js’
