Adjacent sibling

The + combinator matches the second element only if it is immediately following the first element

        

          <div class="my-div">
            <p class="test-p">
              some stuff
            </p>

            <p>
              some other stuff
            </p>
          </div>
        

      
        

          .my-div .test-p + p{
            color : purple;
          }

General sibling

The ~ combinator separates two selectors and matches the second element only if it is preceded by the first, and both share a common parent.

        

          <div class="my-div">
            <p class="test-p">
              some stuff
            </p>

            <p>
              some other stuff
            </p>


            <p>
              yet, even more stuff
            </p>
          </div>
        

      
        

          .my-div .test-p ~ p{
            color : purple;
          }

Child Selector

The > combinator separates two selectors and matches only those elements matched by the second selector that are direct children of elements matched by the first.

        

          <div class="my-div">
            <p>
              some stuff
            </p>

            <p>
              some other stuff
            </p>

            <p>
              yet, even more stuff
            </p>
          </div>
        

      
        

          .my-div > p{
            color : purple;
          }
