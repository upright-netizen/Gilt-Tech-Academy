Gilt Tech Academy 101 - Web Literacy

Course Syllabus

Summer, 2012
Instructor – Nathan Stilwell

Description

Welcome to Gilt Tech Academy 101 - Web Literacy! This class seeks to give you skills to better understand the techniques and languages of the modern web browser. By the end of this class, you will be able to produce web content and layouts from scratch.

Organization

This course will be made up of six (6) 2 hour working sessions that are a mixture of lecture and hands-on excercises. Courses will take place on Monday nights from 6p to 8p in the USA conference room.

Objectives

  1. To give students skills and understanding to transform ideas into the language of the web browser
  2. To offer students options for integrating web skills into their current workflow

Course Topics

HTML

  • Page Structure (doctype, head, body, etc)
  • Structural Markup (headings, section, article, div, etc)
  • Semantic Markup (strong, emphasis, quote, etc)
  • Lists
  • Links
  • Images
  • Tables
  • Forms
  • Video, Audio, and Canvas

CSS

  • Selectors
  • Color
  • Type
  • Background
  • The Box Model
  • Positioning
  • Floating
  • Gradients
  • Transforms
  • Transition
  • Animation

Tools

  • Source Control (Git)
  • Managing open source projects on GitHub
  • Terminal Basics
  • Working with a local web server

Required Materials

*Preferred materials in brackets

  1. HTML & CSS by Jon Ducket isbn: 978–1–118–00818–8
  2. A Computer A Mac
  3. A Browser Google Chrome
  4. A Text Editor Sublime Text 2
  5. An account on GitHub

Grading Plan and Attendence

Just kidding.

There are no grades, tests, or attendance required in this class. Participation is completely optional. That said, there will be assignments. It is highly suggested that students participate in completing excercises and assignments as they are designed to reinforce concepts and enhance learning. Submitting assignments will be done using standard source control technology that will also promote collaboration in real world projects. Time and care have gone into the creation of this course; please apply appropriate time and care when taking it.
